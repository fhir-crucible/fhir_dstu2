Conformance Packages defined as part of FHIR

This specification is a common platform standard that must be adapted to particular use cases. Some particular use cases are common or important enough to be described as a part of the specification itself. These are published a "Conformance Package" - a group of Profiles, Extension Definitions, Value Sets, and examples that are all defined with a common purpose.
