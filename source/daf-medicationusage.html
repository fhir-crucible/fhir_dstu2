DAF Profiles and Usage Guidance for medications

This document provides guidance to implementers and how to use the various profiles related to medications within the context of the DAF Implementation Guide. The list of profiles and their intended usage is as provided in the table below


DAF FHIR profile Purpose Base FHIR Resource
DAF MedicationStatement profile This profile is used to represent active medications summary for a patient. MedicationStatement
DAF MedicationAdministration profile This profile is used to represent active medications summary for a patient. MedicationAdministration
DAF Medication profile This profile is used to represent medication material (e.g. drugs) as part of the other DAF medication related profiles. Medication

DAF medication queries and DAF medication profile usage

  • When QueryRequestors are looking for Active Medications taken by a patient, they should query for DAFMedicationStatement resources.
