Run-time code for FHIR implementation using Java.

In this Package:

  • org.hl7.fhir.instance.model: A set of generated POJO Objects that represent the resources and data types defined in the base FHIR specification. Notable classes: Factory, Constants
  • org.hl7.fhir.instance.formats: Classes to read and write the model objects to/from JSON or XML. Notable Classes: Composer/Parser interfaces, and XmlCompose/Parser, JsonCompose/Parser (other clases are support code)
  • org.hl7.fhir.instance.client: A simple client that makes the FHIR RESTful API even easier to use
  • org.hl7.fhir.instance.utils: Useful utilities for handling Profiles, valuesets, and narrative. Notable Classes: NarrativeGenerator, ValueSetCheckerSimple
  • org.hl7.fhir.instance.validation: Routines to validate Profiles, and instances against Profiles. Notable Clases: InstanceValidator
  • org.hl7.fhir.instance.test:

All the other packages build on the .model package
