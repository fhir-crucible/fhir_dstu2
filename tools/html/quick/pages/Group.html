      • Other Fields
       Field Card. Type and Description
        actual 1..1 boolean
      If true, indicates that the resource refers to a specific group of real individuals. If false, the group defines a set of intended individuals.
        characteristic 0..* Element
      Identifies the traits shared by members of the group.
        characteristic.code 1..1 CodeableConcept
      A code that identifies the kind of trait being asserted.

      Binding: GroupCharacteristicKind(Required)
        characteristic.exclude 1..1 boolean
      If true, indicates the characteristic is one that is NOT held by members of the group.
        characteristic.value[x] 1..1 CodeableConcept | boolean | Quantity | Range
      The value of the trait that holds (or does not hold - see 'exclude') for members of the group.

      Binding: GroupCharacteristicValue(Required)
        code 0..1 CodeableConcept
      Provides a specific type of resource the group includes. E.g. "cow", "syringe", etc.

      Binding: GroupKind(Required)
        identifier 0..1 Identifier
      A unique business identifier for this group.
        member 0..* Patient | Practitioner | Device | Medication | Substance
      Identifies the resource instances that are members of the group.
        name 0..1 string
      A label assigned to the group for human identification and communication.
        quantity 0..1 unsignedInt
      A count of the number of resource instances that are part of the group.
        type 1..1 code
      Identifies the broad classification of the kind of resources the group includes.

      Binding: GroupType(Required)
