Encounter

  • FHIR Implementation: encounter-qicore-qicore-encounter
  • FHIR Parent resource: Encounter
  • An interaction during which services are provided to the patient

    Profile of Encounter for decision support/quality metrics. Defines the core set of elements and extensions for quality rule and measure authors.

    • Key:  = Must support,  = QICore-defined extension,  = Is-Modifier

        location.status 0..1 code
      The status of the participants presence at the specified location during the period specified. If the participant is is no longer at the location, then the period will have an end date/time.
